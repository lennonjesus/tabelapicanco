# Tabela Picanço Web Application

Progressive Web Application (PWA) for discover the best price to buy or sell
your motorcycle.

Based on Bruno Picanço's methodology.

### Get

```
git clone https://github.com/lennonjesus/tabelapicanco.git
```

### Install

```
npm install
```

### Run

```
npm start lite
```
